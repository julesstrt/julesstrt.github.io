Ongoing project based on **LeKiwi**, Hugging Face's open-source mobile manipulator robot.

**Hardware:**
- 3D-printed parts and full mechanical assembly
- Servo motors
- Raspberry Pi 4.0
- Waveshare motor driver board
- Battery and wiring

**Software (ongoing work):**
- Imitation learning for manipulation
- Autonomous navigation using LiDAR and RGB cameras
